# Protein Folding

This is a tutorial on making an AlphaFold like protein folding platform.

## Introduction

AlphaFold is a neural network which is able to [predict protein folding](https://en.wikipedia.org/wiki/Protein_structure_prediction) with variable accuracy. AlphaFold was trained on the content of [PDB](https://en.wikipedia.org/wiki/Protein_Data_Bank). AlphaFold uses a [transformer](https://en.wikipedia.org/wiki/Transformer_(deep_learning)) architecture.
