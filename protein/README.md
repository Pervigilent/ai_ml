# Protein Folding

This is a tutorial on making an AlphaFold like protein folding platform.

## Introduction

AlphaFold is a neural network which is able to [predict protein folding](https://en.wikipedia.org/wiki/Protein_structure_prediction) with variable accuracy. AlphaFold was trained on the content of [PDB](https://en.wikipedia.org/wiki/Protein_Data_Bank). AlphaFold uses a [transformer](https://en.wikipedia.org/wiki/Transformer_(deep_learning)) architecture.

### Other Software

We should also ask ourselves if other software can be leveraged to help in modeling protein folding and interaction. Some possibly relevant software to investigate are the following.

- [NAMD](https://en.wikipedia.org/wiki/NAMD)
- [LAMMPS](https://en.wikipedia.org/wiki/LAMMPS)
- [OpenFOAM](https://en.wikipedia.org/wiki/OpenFOAM)

The third may not be particularly relevant. The first two focus on parallelism and may be a good entry point to develop parallel programming.
