## GPU Acceleration

In this module, we'll look at using GPU compute to accelerate various scientific workloads.

- CUDA
- ZLUDA
- oneAPI
- ROCm
    - HIP
- OpenCL
- SYCL
- Vulkan

We will look at CUDA. Our primary interest is in SYCL and Intel oneAPI for Intel GPUs. Secondarily, we will look at AMDs ROCm and HIP. We will look at the graphics pipeline and ask how both OpenGL and Vulkan might be leveraged as an alternative to these acceleration libraries.

## Introduction

OpenGL is a cross-language, cross-platform API for rendering 2D and 3D vector graphics. OpenGL is no longer updated and has been superseded by Vulkan. As such, we will not concern ourselves with OpenGL.

### oneAPI

OneAPI is an open standard for unified API for accelerators.

### ROCm

ROCm is a software stack (produced by AMD) for GPU programming.

### OpenCL

OpenCL is a framework for creating programs across different platforms. OpenCL specifies both a programming language and an API.

### SYCL

SYCL is a programming model based on C++17.

### Vulkan

Vulkan is a cross-platform API offering an open standard for 3D graphics.
